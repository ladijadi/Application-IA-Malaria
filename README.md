## Installation
1. Clonez le dépôt : `git clone <git@github.com:ladijadi/Application-IA-Malaria.git>`
2. Accédez au répertoire : `cd nom_du_projet`
3. Créez un environnement virtuel : `python -m venv malaria_env`
4. Activez l’environnement virtuel :
   - Linux/Mac : `source env/bin/activate`
   - Windows : `env\Scripts\activate.bat`
5. Installez les dépendances : `pip install -r requirements.txt`
